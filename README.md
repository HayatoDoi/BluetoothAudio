# Bluetooth Audio

LilyGo T5-4.7 Inch S3 E-paper を使用した車載オーディオ

## クイックスタート
1. Visual Studio Code と Python をインストールする
2. Visual Studio Code に PlatformIO というブラグインをインストール
3. Visual Studio Code で本フォルダー開く
4. サードパーティーライブラリーのインストールが完了するまで待機
5. ウィンドウ下部にある ✔ ボタンをクリックしてソースコードをビルドする
6. LilyGo T5-4.7 Inch S3 E-paper をパソコンとUSBで接続する
7. ウィンドウ下部にある → ボタンをクリックしてファームウェアを転送する
